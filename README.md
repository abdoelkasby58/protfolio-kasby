# Abdelrahman Elkasby — Portfolio

> Immersive frontend portfolio built with **Nuxt 3**, **Three.js** & **GSAP**

![Preview](https://img.shields.io/badge/Nuxt-3-00DC82?style=flat&logo=nuxt.js) ![Three.js](https://img.shields.io/badge/Three.js-black?style=flat&logo=three.js) ![GSAP](https://img.shields.io/badge/GSAP-88CE02?style=flat&logo=greensock)

## ✨ Features

- 🌌 **Interactive 3D Scene** — Real-time starfield, nebula clouds, floating rings & animated orb via Three.js
- 🎭 **Scroll Animations** — GSAP ScrollTrigger-powered reveal animations throughout
- 🖱️ **Custom Cursor** — Magnetic cursor with hover states
- 📱 **Fully Responsive** — Works on all screen sizes
- ⚡ **Performance First** — Optimized render loop, lazy imports

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## 🛠️ Tech Stack

| Tech | Purpose |
|------|---------|
| Nuxt 3 | Framework & SSR |
| Three.js | 3D Scene & WebGL |
| GSAP | Animations & ScrollTrigger |
| Space Mono + Syne | Typography |

## 📁 Project Structure

```
cosmos-3d/
├── assets/css/        # Global styles
├── composables/
│   ├── useThreeScene.js    # 3D scene logic
│   └── useGSAPAnimations.js # All animations
├── pages/
│   └── index.vue      # Main portfolio page
└── nuxt.config.ts     # Nuxt configuration
```
